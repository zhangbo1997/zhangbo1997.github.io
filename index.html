We know that "Titanic" was released originally in 1997 and again in 2012. Now, I plotted the global box office returns of both releases on rotation globes .

If we compare both plots, there are interesting facts beyond mere data.

If we drag both globes to US, we can see that both bullet sizes of US box office have similar scales. This is because both data are adjusted using the same logrithmatic function.

Now, if we rotate both globes to Russia, we can see that Russia has a much larger box office of the 2012 release than the 1997 release.
One possible reason behind is that Russia may still be in the effect of post cold war era in 1997, and Russian audience might have psychological repellence against US movie at that time.

Now, if we rotate both globes to China, we can see that in 2012 "Titanic", the box office return in China have grown so much since 1997 that it even supassed the US box office in 2012.
This is a strong indication of the grown influence of Hollywood in China and its huge market for US movies.
